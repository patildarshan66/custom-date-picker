# Custom Date Picker

This is a custom date picker with four preset and six preset group to select date.


For help getting started with Flutter, view the online [documentation](https://flutter.io/).


## Assignment Details

    1. Create a basic flutter app.
    2. Create a pixel-perfect custom date picker widget with preset option, as shown in the [link](https://www.figma.com/file/2p2qBV77ImBXnUqVYoi9yq/Mevolve-Test?t=GhcA0MrekdCTRb55-0).
    3. The home screen should contain three buttons to open three different variants of the custom date picker, as shown in Figma.
    4. Selected dates should be shown below the buttons as shown in Figma.
